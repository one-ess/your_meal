# Сервис доставки бургеров YourMeal. Пет-проект.

Имеется: 

    добавление товаров в корзину.           
    удаление товаров из корзины.
    подсчёт стоимости всех товаров в корзине.
    модальное окно с оформлением заказа, очистка корзины, очистка формы.
    сохранение корзины в localStorage.
    
    
Краткий ввод:
    
    1. Приложение делает fetch запрос на API которое находится по адресу "https://lumbar-animated-resistance.glitch.me"
    2. Сервер отдаёт JSON с данными о бургерах
    3. По запросу /api/product происходит получение списка товаров
    4. По запросу /api/product/{id} происходит получение товара по его id
    5. По запросу /api/product?{list="id,id,id"} происходит получение товаров по id

    Нет зависимостей.
